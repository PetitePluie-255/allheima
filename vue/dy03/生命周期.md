# 生命周期

```
Vue生命周期：一个Vue实例从 创建 到 销毁 的整个过程。
```

### 生命周期四个阶段

```
生命周期四个阶段：
创建阶段 (发送初始化,渲染请求)
挂载阶段 (操作 dom)
更新阶段 (修改数据,更新视图)
销毁阶段 
```

### 生命周期钩子

```
Vue生命周期过程中，会自动运行一些函数，被称为【生命周期钩子】→ 让开发者可以在【特定阶段】运行自己的代码。
```

```
创建阶段 beforeCreate  created
挂载阶段 beforeMount  mounted
更新阶段 beforeUpdate  updated 
销毁阶段 beforeDestroy  destroyed
```
